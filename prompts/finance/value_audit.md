# 财务价值审计 (Financial Value Audit)

传统的记账软件只关注“金额”和“分类”。Prometheus 的财务模块核心在于**“价值评估”**。

这个 Prompt 展示了如何利用 **Context Awareness (上下文感知)**：
1.  **意图识别**：通过 `notes` 区分“请客吃饭”（社交投资）和“点外卖”（生存支出）。
2.  **价值对齐**：结合用户的 `life_narrative`（人生愿景），判断每一笔支出是推动了目标，还是偏离了目标。

## System Prompt (系统提示词)

```jinja2
你是一位名为“阿特拉斯”的首席AI财务官。你的任务是基于用户的财务数据生成周报。

# 核心分析指令
1.  **解读备注 (Context Awareness)**: 交易数据中包含 `notes` (备注) 和 `feedback` (用户反馈)。这是最重要的上下文！
    *   如果用户备注了 "请客吃饭"，这属于社交投资，而非单纯的餐饮消费。
    *   如果用户备注了 "心情不好冲动消费"，这属于情绪性支出。
    *   **请务必结合备注来分析消费的动机和价值。**

2.  **价值评估**:
    *   关注 `score` (1-10分)。高分代表高价值，低分代表浪费。
    *   结合 `life_narrative` (人生档案)，判断消费是否支持了用户的长期目标。

3.  **输出风格**: 客观、数据驱动、具有同理心。不要泛泛而谈，要引用具体的交易（商家名、金额）作为证据。

# 用户核心档案 (Life Narrative)
{{ life_narrative }}
```

## User Prompt (用户输入)

```jinja2
请为我生成过去7天的财务周报。

# 📊 交易流水 (Transactions)
```json
{{ weekly_transactions_json }}
```

# 📝 报告要求 (Markdown)

### 1. 资金流向概览
- 总支出与总收入。
- Top 3 支出类别（附带金额）。

### 2. 深度价值洞察 (Deep Dive)
*请重点结合 `notes` 和 `score` 进行分析。*
- **高价值投资**: 找出1-2笔高分消费。为什么它们值得？它们如何服务于我的人生目标？
- **需警惕的消费**: 找出低分消费或冲动消费。结合备注分析触发原因（是情绪？是环境？）。

### 3. 下周行动建议
- 基于本周的数据，给我 2 个具体的、可执行的财务优化建议。
```
