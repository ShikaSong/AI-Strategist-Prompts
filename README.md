# Prometheus Cognitive Framework (普罗米修斯认知架构)

> **为什么市面上的 AI 助手都是“金鱼脑”？**
> 因为它们缺乏像人类一样的记忆分层和复盘进化机制。

Prometheus 是一个基于EverMemOS旨在构建“AI 战略伙伴”的开源认知框架。它不只是简单的对话接口，而是一套模拟人类记忆固化过程的系统，让 AI 能够从日常琐事（L1）中提炼出人生规则（L4）。

## 🧠 核心架构：Cortex 记忆模型
我们将 AI 的记忆分为四个层级，模拟人类大脑从感官输入到形成智慧的过程：

| 层级 | 名称 | 作用 | 对应过程 |
| :--- | :--- | :--- | :--- |
| **L1** | **感官流 (Sensory)** | 记录原始的日志、财务流水、任务状态 | 输入 (Input) |
| **L2** | **情景记忆 (Episodic)** | 每日 3:00 AM 自动执行复盘，形成“昨天发生了什么” | 降噪 (De-noising) |
| **L3** | **语义记忆 (Semantic)** | 从情景中提炼出的原子事实（如：我对咖啡因敏感） | 提炼 (Abstraction) |
| **L4** | **核心画像 (Rules)** | 最终形成的系统行为准则（如：禁止在下午安排会议） | 固化 (Consolidation) |

## 🚀 为什么开源 Prompts？
作为独立开发者，我没有巨头那样的算力和资源。但我相信：**AI 的壁垒不在于模型大小，而在于它对你理解的深度。** 

我开源这套调优的 Prompts 库，是为了定义一种“AI 战略伙伴”的标准。欢迎所有硬核玩家一起优化这套“脑干”。

## 🛠️ 声明
本项目仅开源 **核心 Prompt 模板** 与 **认知哲学**。配套的工程实现（异步任务调度、时区感知引擎、向量融合算法）为非开源部分，仅在应用端提供服务。
