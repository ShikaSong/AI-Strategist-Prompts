# 贡献指南

感谢你对 Prometheus 认知框架感兴趣。我们非常欢迎对 **Prompts** 和 **认知架构** 的改进建议。

## 如何贡献？
我们不接受随意的代码提交，但我们高度重视对 **Prompt** 的优化。

1. **改进 Prompt**: 如果你发现 `Daily Review` 在处理某些模糊日志时表现不佳，请提交 Pull Request 优化 Jinja2 模板，并附上对比测试结果。
2. **提出新理论**: 如果你认为 L1-L4 架构有缺陷，欢迎在 Issues 中发起关于认知科学的讨论。

## 行为准则
本项目的核心是**“辅助人类成长”**，而非“替代人类思考”。任何试图让 AI 变得通过成瘾性设计操控用户的建议，都将被拒绝。
