# 3AM 进化协议 (The 3AM Evolution Protocol)

## 🌌 核心理念：仿生睡眠 (Biomimicry of Sleep)

在神经科学中，人类并不是在学习的那一刻“记住”知识的。记忆的巩固（Consolidation）主要发生在深度睡眠期间（SWS）。此时，**海马体 (Hippocampus)** 会将白天捕获的临时、混乱的短期记忆，回放并传输给 **大脑皮层 (Neocortex)**，从而转化为长期、稳定的语义记忆。

Prometheus 框架复刻了这一生物学过程。

我们拒绝让 AI 在每次对话中都进行高消耗的全量检索。相反，我们设计了 **“3AM 进化协议”** —— 一个异步的、深度推理的批处理过程。

## ⚙️ 为什么选择异步 (Why Asynchronous)?

### 1. 降噪 (De-noising)
白天的 **L1 感官流**（Sensory Stream）充满了噪音：随手记的碎碎念、临时的情绪波动、琐碎的流水账。只有在一天结束后，AI 才能以上帝视角（God View）审视全天，区分哪些是**“信号”**，哪些是**“噪音”**。

### 2. 系统二思维 (System 2 Thinking)
丹尼尔·卡尼曼提出，人类有“快思考”和“慢思考”。
- **日间交互**：使用快模型（Fast Model），响应迅速，处理战术问题。
- **夜间复盘**：使用慢模型（Smart Model），进行深度的因果推演（Reasoning），处理战略问题。

### 3. 成本与性能
通过将重计算任务移至凌晨，我们避免了用户在线时的 API 等待时间，并能集中算力处理复杂的逻辑仲裁。

## 🔄 协议执行管线 (The Pipeline)

该协议通常在用户当地时间 **03:00 AM** 触发，包含四个阶段：

1.  **提取 (Extraction)**
    *   扫描昨日所有的日志、任务反馈、财务流水、生物数据。
    *   *Input*: "喝了冰美式手抖" + "下午任务失败"。
2.  **推演 (Foresight)**
    *   寻找跨数据的因果关系。
    *   *Insight*: "摄入高浓度咖啡因" -> 导致 -> "生理性焦虑" -> 导致 -> "执行力崩溃"。
3.  **仲裁 (Arbitration)**
    *   检测新发现的事实是否与旧规则冲突。
    *   *Logic*: 基于时间近因效应 (Recency) 解决冲突，更新 L3/L4 记忆库。
4.  **指令 (Directives)**
    *   生成次日的 **Planner Directives**。
    *   *Output*: "今日禁止安排 10:00 后的咖啡会议，并强制插入 20 分钟冥想。"
