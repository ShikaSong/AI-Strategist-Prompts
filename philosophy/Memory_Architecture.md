# L1 - L4 记忆演进哲学

### L1: 原始数据的混沌
AI 伙伴的第一步是感知。不仅仅是对话，还包括财务支出（你的价值观体现）、心率睡眠（你的精力基座）和随手记（你的潜意识波动）。

### L2: 情景化的叙事
人类在睡眠时，大脑会重放一天的记忆。Prometheus 的 `Daily Evolution Protocol` 在每日凌晨 3 点启动，它通过 Prompts 强制 AI 扮演“观察者”，对 L1 数据进行非线性审计，生成 L2 日志。

### L3: 原子事实的提炼
通过 `Semantic Memory Layer`，我们将 L2 中的信息打碎为“原子事实”。每一个事实都带有证据链，例如：
- [Fact 105]: 用户在处理财务报表时表现出极高的拖延倾向。
- [Evidence]: 连续 3 天计划失败，且心率在任务时段上升。

### L4: 核心规则的固化
这是最终阶段。AI 不再是建议你做什么，而是系统级地约束自己。它形成 `User Manual`，直接指导下一周的 Planner 引擎。
